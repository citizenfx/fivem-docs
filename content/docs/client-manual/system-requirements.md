To run FiveM, RedM or FiveM for GTAV Enhanced, your system must meet the minimum requirements of the underlying game.

All platforms require a _fully_ updated version of Windows 10 or Windows 11. An outdated operating system may not
work. It is generally advised to upgrade to Windows 10 or 11 when you can as this provides the best experience.

## FiveM

To [run FiveM][installing], your system must meet the [minimum requirements of the original game][gtav-system-specs].

|                 | Minimum                                                 | Recommended                                         |
|-----------------|-----------------------------------------------------------|-------------------------------------------------------|
| OS              | Windows 10 64-bit                                       | Windows 10 64-bit                                   |
| CPU             | Intel Core 2 Q6600 @ 2.40GHz / AMD Phenom 9850 @ 2.5GHz | Intel Core i5 3470 @ 3.2GHz / AMD X8 FX-8350 @ 4GHz |
| GPU<sup>1</sup> | NVIDIA 9800 GT 1GB / AMD HD 4870 1GB / Intel HD GT2     | NVIDIA GTX 660 2GB / AMD HD7870 2GB                 |
| RAM             | 8GB (4 may work)                                        | 16GB                                                |
| HDD<sup>2</sup> | 125GB + ~4GB                                            | 125GB + 8GB                                         |

_Notes:_

1. May not work with some older AMD laptop GPUs. More on that [here][client-issues-gpu].
2. 125GB for the original game + additional FiveM cache.

## FiveM for GTAV Enhanced

FiveM for GTAV Enhanced runs on top of [Grand Theft Auto V Enhanced][gtav-system-specs].

|                 | Minimum                                             | Recommended                                            |
|-----------------|------------------------------------------------------|-----------------------------------------------------------|
| OS              | Windows 10 (build 1909 or above)                    | Windows 11                                              |
| CPU             | Intel Core i7-4770 / AMD FX-9590                    | Intel Core i5-9600K / AMD Ryzen 5 3600                  |
| GPU             | NVIDIA GTX 1630 4GB / AMD RX 6400 4GB               | NVIDIA RTX 3060 8GB / AMD RX 6600 XT 8GB                |
| RAM             | 8GB                                                 | 16GB (dual-channel)                                     |
| Storage         | 105GB (SSD required)                                | 105GB (DirectStorage-compatible drive)                  |

## RedM

RedM requires a [fully updated installation of Red Dead Redemption 2][where-to-buy-rdr3], which must meet the [minimum requirements of the original game][rdr2-system-specs].

|                 | Minimum                                     | Recommended                                        |
|-----------------|------------------------------------------------|-------------------------------------------------------|
| OS              | Windows 10 64-bit                           | Windows 10 64-bit                                  |
| CPU             | Intel Core i5-2500K / AMD FX 6300           | Intel Core i7-4770K / AMD Ryzen 5 1500X            |
| GPU             | NVIDIA GTX 770 2GB / AMD Radeon R9 280 3GB  | NVIDIA GTX 1060 6GB / AMD Radeon RX 480 4GB        |
| RAM             | 8GB                                         | 12GB                                                |
| Storage         | 150GB                                       | 150GB                                               |
