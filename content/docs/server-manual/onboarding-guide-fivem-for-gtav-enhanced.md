## Getting the Server Files

Server files for FiveM for GTAV Enhanced can be downloaded from the [Server Download](https://docs.fivem.net/docs/server-download/) page.

## Converting Assets for FiveM for GTAV Enhanced

We published Alchemist, a tool to convert existing assets to work with FiveM for GTAV Enhanced.
For more information on how to use Alchemist, please refer to the [Alchemist Documentation](https://docs.fivem.net/docs/alchemist/).

## Differences Between FiveM and FiveM for GTAV Enhanced

We created a page that details the differences between FiveM and FiveM for GTAV Enhanced, including removed features, breaking changes, new config variables, and behavioral differences. You can find this information on the [What's Changed in FiveM for GTAV Enhanced](https://docs.fivem.net/docs/developers/legacy-vs-enhanced/) page.
