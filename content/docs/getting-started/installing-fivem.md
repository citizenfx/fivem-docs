If you already have FiveM downloaded and installed, you can skip this step.
{{% /alert %}}

{{% video src="https://runtime.fivem.net/edu/step-1_install-setup-fivem.mp4" %}}

### 1. Download the FiveM client

1. Go to [**FiveM.net**](https://fivem.net/) and click **Download Client**.
2. Review the Creator Platform License Agreement.
3. If you agree, click **Sure, I Agree!** to download the client.

### 2. Install and set up FiveM client

1. Go to your **Downloads** folder and run **FiveM.exe**.
2. When prompted, navigate to your GTAV installation folder and select GTA5.exe.
3. Click **Open** to confirm the path.

### 3. Update game data

A message appears stating that local game data needs to be updated.
Click Yes. FiveM prepares the launch of the data. This can take several minutes.

### 4. Sign in with Cfx.re

1. Review the **Terms of Service**.
2. If you agree, click **I Accept** to continue.\
   A new window opens prompting you to sign in with Cfx.re. Don't skip this!
3. Click **Sign in with Cfx.re**.\
   Your browser opens.
4. Click **Authorize** to grant access.
5. Go back to FiveM and a **Welcome Username** appears.
6. Click **Continue**.

Great! FiveM is now installed and ready to use. Next, let's set up your own local server!
